# AI-MBTI 网页小游戏合集  
# AI-MBTI Web Game Collection

## 中文说明

这是一个用于存放各种脑洞型网页小游戏的项目仓库。

项目最初从一个「AI 模型适配测试」开始：受 MBTI 类趣味测试启发，通过选择题的形式，让用户获得一个娱乐向的 AI 模型适配结果。后续本仓库也会继续收录更多轻量级、可直接打开运行的网页小游戏、互动测试、信息卡片和创意实验页面。

本项目更偏向创意表达和网页互动实验，不追求严肃测评、学术评价或商业推广。

## 当前内容

### AI 模型适配测试

一个 30 题的网页测试游戏。  
用户根据自己的任务偏好、信息处理方式、工作节奏、风险敏感度和使用场景进行选择，最终生成一个娱乐向的 AI 模型适配结果。

功能包括：

- 30 道选择题
- 点击选项后自动进入下一题
- 支持返回上一题重新选择
- 生成模型适配结果
- 显示适配度和模型排行
- 支持复制分享文案
- 支持重新测试和返回首页

## 使用方式

直接下载或打开 HTML 文件即可运行。

不需要后端。  
不需要数据库。  
不需要安装依赖。

如果部署到 GitHub Pages，也可以作为静态网页直接访问。

## English Description

This repository is a collection of experimental web-based mini games and interactive creative pages.

The project started with an “AI Model Matching Test” inspired by MBTI-style personality quizzes. Through a set of multiple-choice questions, users receive an entertainment-oriented AI model matching result. More lightweight web games, interactive tests, idea prototypes, and creative HTML pages may be added to this repository in the future.

This project focuses on creative interaction and web-based experimentation. It is not intended for academic evaluation, formal benchmarking, or commercial promotion.

## Current Project

### AI Model Matching Test

A 30-question interactive web quiz.

Users answer questions based on their task preferences, information-processing habits, workflow style, risk sensitivity, and usage scenarios. The page then generates an entertainment-oriented AI model matching result.

Features include:

- 30-question quiz flow
- Automatic progression after selecting an answer
- Previous-question return and answer correction
- AI model matching result page
- Match score and model ranking
- Shareable result text
- Restart and homepage return options

## How to Use

Open the HTML file directly in a browser.

No backend required.  
No database required.  
No dependencies required.

The page can also be deployed directly through GitHub Pages as a static website.
